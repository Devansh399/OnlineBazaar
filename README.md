
# 🛒 OnlineBazaar – Full Stack MERN E-Commerce Platform

OnlineBazaar is a modern full-stack e-commerce web application built using the MERN stack. It provides a complete online shopping experience with secure user authentication, product management, shopping cart, order processing, payment integration, and an admin dashboard.

## ✨ Features

### 👤 User Features

* User Registration & Login with JWT Authentication
* Browse and Search Products
* Product Details Page
* Shopping Cart using Redux Toolkit
* Quantity Management
* Secure Checkout Process
* Razorpay Payment Gateway Integration
* Order Confirmation via Email
* User Profile Management
* Order History

### 🛠️ Admin Features

* Admin Authentication & Authorization
* Dashboard with Analytics
* Add, Edit & Delete Products
* Manage Orders
* Manage Users
* Product Inventory Management

## 💳 Payment Integration

* Razorpay Test Mode Integration
* Payment Verification
* Automatic Order Creation after Successful Payment
* Student Bypass Mode for testing without payment (development only)

## 📊 Admin Dashboard

* Total Users
* Total Products
* Total Orders
* Total Revenue
